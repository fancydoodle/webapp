How to use

t Activate Text mode
Escape Return to Draw mode

Text Mode

ctrl + [ Rotate Text Left
ctrl + ] Rotate Text Right

→ Text blocks are draggable & editable ←

Right Click Delete text

cmd+ cmd- Resize Text or Line

Draw Mode

b Black

w White

y Yellow

r Red

Hold:
shift Straight lines
alt Dashed lines
cmd/ctrl Circles
Right Click Bezier curves
cmd + z Undo
cmd + y Redo

All app data is stored in your browser.
