How to use

Text Mode

t - Activate text mode

Escape - Return to Draw mode

→ Text blocks are draggable & editable ←

** Right click to delete text **

cmd + cmd - Resize text or line

Draw Mode

b Black

w White

y Yellow

r Red

Hold shift to create straight lines

Hold alt to create dashed lines

Hold cmd/ctrl to draw circles

Right Click to create Bezier curves

cmd + z Undo

cmd + y Redo

All app data is stored on this computer only.
